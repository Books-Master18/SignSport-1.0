Добро пожаловать на SignSport!

Подберём подходящий для вас вид спорта не по вашей физической форме, а по личностным особенностям, который определим используя для этого анализ вашего почерка!

Нейросеть SignSportNeuralNetwork 1.0 (сокращенно 2S2N 1.0) использует результаты графологического анализа почерка (то есть картину вашего характера и личности) для определения вида спорта, который подходит вашему характеру — а не только физическим данным. Это уникальный подход, основанный на научных связях между почерком, личностью и предрасположенностью к определённым видам физической активности.

Этапы работы проекта:

  • Проанализировать почерк человека
  • Загрузить данные его характера в нейросеть
  • Получить долгожданный результат!

Загрузите готовый отчёт по анализу вашего почерка — и получите рекомендацию по подходящему виду спорта прямо сейчас!
